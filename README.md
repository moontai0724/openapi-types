## Install

### NPM

```bash
npm install -d @moontaiworks/openapi-types
```

### Yarn

```bash
yarn add -d @moontaiworks/openapi-types
```

### PNPM

```bash
pnpm add -d @moontaiworks/openapi-types
```

## Usage

```typescript
import { add, divide, max } from "@moontaiworks/openapi-types";

add(1, 2); // 3
divide(1, 2); // 0.5
max(1, 2); // 2
```
